# CS2_LosingTeamSlayer
Плагин, который автоматически казнит (slay) всех живых игроков проигравшей команды по окончании раунда. Причина определяется по событию окончания раунда (взрыв/обезвреживание бомбы, освобождение/гибель заложников и т.д.). Перед казнью можно задать задержку. Также плагин защищает террористов от взрыва бомбы (восстанавливает здоровье)

### Требования
~~~
CounterStrikeSharp API версии 369 или выше
.NET 10.0 Runtime
~~~
### Конфигурационные параметры
~~~
css_losingteamslayer_enabled <0/1> def. 1 – Включает или выключает работу всего плагина. При значении 0 плагин полностью отключён, казни не производятся.
css_losingteamslayer_slaymessage <0/1> def. 1 – Определяет, будет ли в чат выводиться сообщение с указанием причины казни (например, «Не пускай КТ к заложникам!»). При 0 сообщения не выводятся.
css_losingteamslayer_delay <0.0-10.0> def. 2.0 – Задаёт время задержки (в секундах) между окончанием раунда и моментом казни проигравшей команды. Позволяет игрокам увидеть итог раунда до применения казни.
~~~
### Консольные команды
~~~
css_losingteamslayer_enabled <0/1> – Включить (1) или выключить (0) плагин
css_losingteamslayer_slaymessage <0/1> – Включить (1) или выключить (0) вывод в чат сообщения о том, за что казнена команда
css_losingteamslayer_delay <число> – Установить задержку перед казнью (в секундах, от 0.0 до 10.0)
~~~
